# CLAUDE.md

## O projektu

Tento repozitář obsahuje jednoduché HTML nástroje bez build procesu.

## Soubory

- `rabaty.html` — přehled rabatů pro dealery (Simply). Čistý HTML/CSS/JS, data v localStorage.

## Pravidla

- Pouze čistý HTML, CSS, JavaScript — žádné frameworky, žádný build
- Vše v jednom souboru (HTML + CSS + JS dohromady)
- Responzivní design (funguje na mobilu i počítači)
- Data se ukládají do localStorage prohlížeče
